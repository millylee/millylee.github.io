博客历程

2012 年在折腾 Github Pages 时看到阮一峰写的一句话，觉得非常贴切。

喜欢写 Blog 的人，会经历三个阶段。
第一阶段，刚接触Blog，觉得很新鲜，试着选择一个免费空间来写。
第二阶段，发现免费空间限制太多，就自己购买域名和空间，搭建独立博客。
第三阶段，觉得独立博客的管理太麻烦，最好在保留控制权的前提下，让别人来管，自己只负责写文章。

我猜任何技术宅又有写博客习惯的人应该没少折腾过吧，这点从网络上一堆折腾笔记就可以看出。

从今天起，我会坚持用 Github 来写博客，让其它平台都见鬼去吧，以下是我的博客 Timeline。

  • 2008 年，以网易博客为主的 BSP 博客
  • 2009 年，使用 WordPress 搭建了独立博客
  • 2011 年，开始使用点点网
  • 2012 年，Github Pages + Jeklly 托管静态博客
  • 2013 年，对比了简书、SegmentFault 等专一、简洁的写作功能
  • 2014 年，对比了 Ghost、Farbox、Hexo 等写作平台或工具

最终选择了 Hexo，与 Jeklly 相比，配置容易、生成效率高。

心态转变

折腾并没有什么不好，也算是一种兴趣吧，不过现在开始慢慢觉得时间不够用，所以希望将时间放在有用的事情上面。今年由于公司大变动，有机会从前端转到自己另一个喜欢的岗位——产品汪。

该博客也将用来记录我的成长，希望两、三年后会有所收获。

致未来

花时间折腾不如花同等时间写作。
